# Social Links Profile

A responsive profile card built for the Frontend Mentor **Social links profile** challenge.

This project uses plain HTML and CSS to recreate a dark-themed profile card with a profile image, name, location, short bio, and a list of social buttons.

---

## Demo

Open `index.html` in your browser or use a local development server to see the profile card.

---

## Features

- Responsive card layout for mobile screens
- Centered profile card with a dark background
- Rounded profile image and card container
- Styled social links with hover and focus states
- Accessible focus ring for keyboard users
- Clean typography using the Inter font

---

## Built with

- HTML5
- CSS3
- Frontend Mentor challenge template

---

## Project structure

- `index.html` — page markup and content
- `style.css` — styles for layout, typography, and interactive states
- `assets/` — image and font assets used by the project

---

## How to use

1. Clone or download the project repository.
2. Open `index.html` in a browser.
3. If you want to preview changes while editing, use a local server or VS Code Live Server.

---

## What I learned

- How to use Flexbox to center a card layout on the page.
- How to build a responsive container using `width: min(330px, 100%)`.
- The importance of matching CSS selectors exactly to HTML class names.
- How to create accessible hover and focus states for interactive links.
- How to organize styles for typography, spacing, and button design.

---

## Notes

- The design is based on the challenge preview and the included style guide.
- The current version is optimized for a mobile-first layout.
- You can expand the project with a desktop layout or add more social links if desired.
